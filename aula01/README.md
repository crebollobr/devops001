# Introdução à Computação e Uso Básico do Computador

Este material foi preparado para alunos iniciantes, com pouco ou nenhum conhecimento prévio em informática. O objetivo é apresentar, de forma simples e organizada, os conceitos básicos sobre computadores, sistemas, internet e o uso inicial do terminal (linha de comando).

---

## 1. Áreas e Perfis Profissionais em Computação

Antes de falar de tecnologia, é importante entender que existem diferentes áreas de atuação na informática:

* **Profissionais de TI**: pessoas que trabalham com tecnologia da informação.
* **Desenvolvedor**: cria programas, sites e aplicativos.
* **Infraestrutura**: cuida de servidores, redes, sistemas e equipamentos.
* **DevOps**: integra desenvolvimento e infraestrutura para facilitar entregas e manutenção.
* **DevSecOps**: adiciona segurança ao processo de desenvolvimento e operação.

Essas áreas costumam trabalhar juntas dentro de empresas.

---

## 2. Segurança da Informação e LGPD

* **Segurança da Informação**: conjunto de práticas para proteger dados e sistemas.
* **LGPD (Lei Geral de Proteção de Dados)**: lei brasileira que protege os dados pessoais das pessoas.

Exemplos simples de boas práticas:

* Não compartilhar senhas
* Desconfiar de links desconhecidos
* Usar senhas fortes

## 3. Conceitos Básicos de Hardware

O computador é formado por partes físicas chamadas **hardware**:

* **CPU**: é o “cérebro” do computador, onde os cálculos acontecem.
* **RAM**: memória temporária usada enquanto o computador está ligado.
* **HD / SSD**: onde os arquivos ficam armazenados.

### Arquitetura de von Neumann

Modelo básico de funcionamento dos computadores modernos:

* Processador
* Memória
* Entrada e saída de dados

Leitura recomendada:

* [https://pt.wikipedia.org/wiki/Arquitetura_de_von_Neumann](https://pt.wikipedia.org/wiki/Arquitetura_de_von_Neumann)

## 4. Computador, Sistemas e Programas

* **Computador**: máquina física.
* **Sistema**: conjunto de programas que fazem o computador funcionar.
* **Sistema Operacional**: programa principal (Windows, Linux, macOS).
* **Aplicativo**: programa para o usuário (navegador, editor de texto).

### Modelo Cliente-Servidor

* **Cliente**: computador ou aplicativo do usuário.
* **Servidor**: computador que fornece serviços (sites, email, arquivos).

Exemplos de serviços:

* Sites
* Email
* Armazenamento de arquivos

## 9. Comandos Básicos no Terminal (CMD)

Comandos iniciais para conhecer o ambiente:

* `dir` → lista arquivos
* `dir /?` → ajuda do comando
* `dir /o:d` → organiza por data
* `cd` → entra em pastas
* `echo hello` → escreve um texto
* `echo hello > arquivo.txt` → cria arquivo com texto
* `type arquivo.txt` → mostra conteúdo do arquivo
* `del arquivo.txt` → apaga arquivo
* `copy` → copia arquivos
* `move` → move arquivos
* `ren` → renomeia arquivos

Esses comandos ajudam a entender como o sistema funciona internamente.
